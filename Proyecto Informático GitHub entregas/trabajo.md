    <h1><b>Comandos basicos de HTML</b></h1>

    <h2><b>¿Que son?</b></h2>
    
    <p>
      Los "comandos" en HTML se conocen técnicamente como etiquetas (tags). Estas estructuras le dicen al navegador cómo organizar y mostrar el contenido de una página web.
    </p>
    
    <p>
      A continuación, se presentan las etiquetas fundamentales divididas por su función:
    </p>

    <p><b>1. Estructura Básica del Documento</b></p>
    <p>Estas etiquetas son indispensables para que cualquier archivo HTML funcione correctamente:</p>

    <p><b>html:</b> Envuelve todo el contenido de la página.</p>
    <p><b>head:</b> Contiene información "invisible" para el usuario.</p>
    <p><b>title:</b> Define el título que aparece en la pestaña del navegador.</p>
    <p><b>body:</b> Contiene todo lo que el usuario ve.</p>
 
    <p><b>2. Formato de Texto</b></p>
    <p><b>h1 a h6:</b> Encabezados de diferentes niveles.</p>
    <p><b>p:</b> Define un párrafo de texto.</p>
    <p><b>br:</b> Inserta un salto de línea sencillo.</p>
    <p><b>strong o b:</b> Aplican formato de negrita.</p>
    <p><b>em o i:</b> Aplican formato de cursiva.</p>

    <p><b>3. Listas, Enlaces y Multimedia</b></p>
    <p><b>a:</b> Crea un hipervínculo (enlace).</p>
    <p><b>img:</b> Inserta una imagen.</p>
    <p><b>ul y ol:</b> Crean listas con viñetas o números.</p>
    <p><b>li:</b> Define cada elemento de una lista.</p>

    <p><b>4. Contenedores y Organización</b></p>
    <p><b>div:</b> Un contenedor genérico para agrupar elementos.</p>
    <p><b>span:</b> Un contenedor para pequeñas secciones de texto.</p>
